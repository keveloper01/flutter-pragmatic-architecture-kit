# Flutter Feature Delivery Skill

Use this skill when implementing a feature slice.

## Delivery Order

1. Model
2. Mapper
3. Provider
4. Repository
5. BLoC
6. Page and widgets
7. Tests

## Implementation Standard

- Keep constructors explicit.
- Keep nullable values intentional.
- Use clear failure paths.
- Prefer small widgets with obvious inputs.
- Keep BLoC state easy to inspect in tests.

## Done Means

- The feature follows the architecture.
- The main behavior works.
- At least one failure case is tested.
- Analysis is clean.
