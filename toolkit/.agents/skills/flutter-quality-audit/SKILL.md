# Flutter Quality Audit Skill

Use this skill when validating a Flutter change.

## Audit Areas

- Architecture boundaries
- State coverage
- Error handling
- Test quality
- Naming and folder shape
- Async behavior

## Commands

Run the project-appropriate analysis and tests. Prefer targeted tests first, then broaden when shared code changed.

## Report Format

1. Findings by severity.
2. Commands run.
3. Test gaps.
4. Recommended next action.
